# Contribuer à homelab-vlan-refactor-configs

Merci de votre intérêt pour ce projet ! Voici comment contribuer.

## Comment contribuer

1. Forkez le dépôt
2. Créez une branche (`git checkout -b feature/ma-fonctionnalite`)
3. Committez vos changements (`git commit -m "feat: ajout de..."`)
4. Pushez vers la branche (`git push origin feature/ma-fonctionnalite`)
5. Ouvrez une Pull Request

## Conventions de Commit

- `feat:` Nouvelle fonctionnalité
- `fix:` Correction de bug
- `docs:` Documentation
- `style:` Formatage (sans impact fonctionnel)
- `refactor:` Refactoring de code
- `test:` Ajout de tests
- `chore:` Maintenance (deps, build, etc.)

## Code Review

- Toutes les PR doivent être reviewées
- Les tests doivent passer
- La documentation doit être mise à jour

## Questions ?

Ouvrez une issue ou contactez le mainteneur.
